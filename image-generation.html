Image Generation AI

Gemini 2.5 Flash Image 모델을 사용하여 환자의 치아 사진에 라미네이트 시술 결과를 실시간으로 생성합니다. 얼굴은 그대로 유지하면서 치아만 변환하는 의료 시뮬레이션 AI입니다.

핵심 기술

🧠
Gemini 2.5 Flash Image

Google의 최신 멀티모달 AI 모델로, 이미지를 입력받아 수정된 이미지를 직접 생성합니다. 기존 이미지 필터 방식과 달리 치아의 형태, 정렬, 색상을 완전히 재설계할 수 있습니다.

🎨
프롬프트 엔지니어링

3가지 스타일(자연형/프리미엄형/화이트형)마다 최적화된 프롬프트를 사용합니다. 각 프롬프트는 치과 심미 기준에 맞춰 설계되었습니다.

자연형 (Natural)

VITA A2 수준 밝기. 자연스러운 정렬 개선. "원래 좋은 치아" 느낌. 확실한 변화이지만 인공감 없음.

프리미엄형 (Premium)

VITA A1 수준 밝기. 이상적 비율의 정밀 정렬. 포슬린 질감의 매끄러운 광택. 최고급 시술 결과.

화이트형 (White)

VITA B1/Bleach 수준 순백. 완벽한 직선 배열. 할리우드 셀럽 스마일. 극적인 변화.

채택

🔒
보안 및 개인정보

  • API 키는 브라우저 localStorage에만 저장 (코드에 포함하지 않음)
  • 환자 이미지는 Google Gemini API로 전송 후 즉시 폐기 (Google 정책)
  • 생성된 결과 이미지는 Supabase DB에 암호화 저장
  • GitHub 등 공개 저장소에 API 키가 노출되면 자동 차단됨
