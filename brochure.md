## ❓ 자주 묻는 질문

### Q1: AI 시뮬레이션의 정확도는 얼마나 되나요?
**A:** 92%의 정확도로 실제 시술 결과와 매우 유사합니다. 의료 AI 기술 기반으로 색상, 배열, 대칭을 정밀 분석합니다.

### Q2: 도입 시 직원 교육이 필요한가요?
**A:** 아니요. 직관적인 UI로 설계되어 짧은 설명만으로 바로 사용 가능합니다. 매뉴얼 영상 제공.

### Q3: 환자 정보 보안은 안전한가요?
**A:** Supabase 의료 규제 준수 DB 사용. 환자 정보는 암호화되어 안전하게 보관됩니다.

### Q4: 초기 도입 비용은 얼마인가요?
**A:** 상담 전환율 증가로 3~6개월 내 투자비 회수 가능한 구조입니다. 자세한 가격은 담당자와 상담하세요.

### Q5: 기존 환자 데이터 마이그레이션이 가능한가요?
**A:** 네, 기존 환자 정보를 DB로 자동 마이그레이션해드립니다.

### Q6: 다국어 지원이 되나요?
**A:** 현재 한국어 기반입니다. 향후 다국어 확장 예정.

### Q7: 모바일 환자용 앱이 있나요?
**A:** 웹 기반으로 별도 앱 설치 없이 모든 기기에서 사용 가능합니다.
