MediaPipe Face Mesh

Google MediaPipe의 Face Mesh 솔루션은 단일 카메라 이미지에서 468개의 3D 얼굴 랜드마크를 실시간으로 추정합니다. Smile Vision AI의 치아 탐지 파이프라인의 첫 번째 핵심 엔진입니다.

468 Landmarks 기술

🧠
Face Mesh 개요

MediaPipe Face Mesh는 머신러닝 파이프라인을 통해 468개의 3D 얼굴 표면 랜드마크를 실시간으로 추정합니다. 얼굴의 윤곽, 눈, 코, 입, 눈썹 등 세밀한 영역까지 정밀하게 매핑하며, 특히 refineLandmarks: true 옵션을 활성화하면 입술과 눈 주변 영역이 더욱 정밀해집니다.

📐 랜드마크 구성

  • 얼굴 윤곽 (Face Oval): ~36개
  • 눈 영역 (Eyes): ~32개
  • 눈썹 (Eyebrows): ~20개
  • 코 (Nose): ~18개
  • 입 (Lips): ~40개
  • 홍채 (Iris, refined): +10개
  • 기타 표면 포인트: ~322개

⚙ 동작 원리

  • Stage 1: BlazeFace로 얼굴 위치 감지
  • Stage 2: 감지된 얼굴 영역 크롭
  • Stage 3: Mesh 모델로 468포인트 추정
  • 각 포인트는 (x, y, z) 3D 좌표
  • x, y: 0~1 정규화 / z: 상대적 깊이

치아 탐지를 위한 22개 내입술 랜드마크

🩻
Inner Lip Landmarks

468개 랜드마크 중 치아 탐지에 핵심적으로 사용되는 22개 포인트입니다. 상부 내입술 라인과 하부 내입술 라인이 만드는 폐곡선 내부가 치아 노출 영역이 됩니다.

구분 랜드마크 인덱스 개수 역할
상부 내입술 78 191 80 81 82 13 312 311 310 415 308 11개 상부 치아 경계선 (윗입술 안쪽)
하부 내입술 308 324 318 402 317 14 87 178 88 95 78 11개 하부 치아 경계선 (아랫입술 안쪽)
// 치아 영역 폴리곤 생성
const upperInner = [78, 191, 80, 81, 82, 13, 312, 311, 310, 415, 308];
const lowerInner = [308, 324, 318, 402, 317, 14, 87, 178, 88, 95, 78];

// 상부 + 하부 = 폐곡선 (치아 노출 영역)
const teethRegion = [
  ...upperInner.map(i => [lm[i].x * W, lm[i].y * H]),
  ...lowerInner.map(i => [lm[i].x * W, lm[i].y * H])
];

// Canvas 클리핑으로 치아 마스크 생성
ctx.beginPath();
teethRegion.forEach((pt, i) => {
  i === 0 ? ctx.moveTo(pt[0], pt[1]) : ctx.lineTo(pt[0], pt[1]);
});
ctx.closePath();
ctx.clip();

상부와 하부 랜드마크는 양쪽 끝에서 공유 포인트(78, 308)를 가지며, 이를 연결하면 입 벌림에 따라 동적으로 변하는 치아 노출 영역이 자동으로 계산됩니다.

const faceMesh = new FaceMesh({
  locateFile: (file) => `https://cdn.jsdelivr.net/npm/@mediapipe/face_mesh@0.4/${file}`

  maxNumFaces: 1,

});

Smile Vision AI에서의 활용

🚀
파이프라인 통합

MediaPipe Face Mesh는 Smile Vision AI의 4-Stage Pipeline에서 Stage 1(Face Detection)과 Stage 2(Teeth Segmentation)의 핵심 엔진으로 사용됩니다.

Stage 1 - 얼굴 감지

환자가 업로드한 치아 사진에서 얼굴을 감지하고 468개 랜드마크를 추출합니다. 얼굴이 감지되지 않으면 색상 기반 폴백 알고리즘으로 자동 전환됩니다.

Stage 2 - 치아 영역 분리

22개 내입술 랜드마크로 생성한 폴리곤 내부에서 구강(어두운 영역)과 잇몸(붉은 영역)을 제거하고, 순수 치아 픽셀만 분리한 마스크를 생성합니다.

AI 이미지 생성 가이드

추출된 치아 영역 정보를 기반으로 Gemini AI에게 정확한 변환 영역을 지시합니다. "이 영역만 라미네이트로 변환하라"는 맥락을 제공합니다.

Dental Scoring

치아 영역의 밝기, 정렬, 대칭성 등을 분석하여 6가지 항목의 치아 점수를 산출합니다. 랜드마크 좌표를 기반으로 정렬도와 대칭도를 계산합니다.
